# Build Mini Apps on Startale

## Overview

Welcome to the **Startale Superstars Incubation technical documentation**. This comprehensive guide will help you understand how to build a Mini App that meets the requirements of our incubation program. The guide will explain everything from the basic concepts of Mini Apps and technical structure to creating a usable MVP using Farcaster's Mini Apps framework.

All of this in order to establish the foundations that will allow you to launch your Mini Apps on the Startale App in the future.
